Функціональні можливості доступності ACET

Функції та можливості програми ACET розроблені для підтримки користувачів з обмеженими можливостями. Дотримання відповідних стандартів галузі дозволяє використовувати функції доступності, які вбудовані в операційну систему Windows та архітектуру .NET. Ці вбудовані можливості дозволяють забезпечити відповідність вимогам Розділу 508 Закону про реабілітацію США.

Примітка. Функціональність діаграми та аналізу CSET не стала доступною або не відповідає вимогам розділу 508.

Діаграма: У діаграмі використовується розширений список комбінацій клавіш.

Аналіз: Доступність для аналізу функціональності може бути досягнута шляхом друку звітів або створення їх версії для відображення на екрані. Звіти генеруються у форматі HTML.

Функція високого контрасту зображення

Всі текстові області в ACET, крім сторінок Діаграма та Аналіз, підтримують переключення в режим високого контрасту зображення в операційній системі Windows.

Доступ з клавіатури

ACET можна використовувати за допомогою клавіатури. Всі області програми, крім сторінки Аналіз, можна відкрити за допомогою клавіатури. Більшість доступу з клавіатури реалізовано за допомогою клавіш TAB та стрілок для навігації, клавіш SPACE та ENTER для вибору. Додаткові скорочені команди або гарячі клавіші можна знайти в діалозі Клавівтурні скорочення.
