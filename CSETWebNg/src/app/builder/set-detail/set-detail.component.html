Деталі модулю

Модуль може бути заснований на галузевому стандарті або може бути просто набором питань. Модулі, засновані на стандартах, зазвичай проходять шлях вимог, щоб передати вимоги до набору.

Якщо ви створюєте стандарт, керуйте вимогами, натиснувши . Шлях покаже структуру відповідних запитань, коли у «режимі питань» оцінювання.

Рекомендується додавати будь-які пов’язані документи до модулю. Натисніть кнопку для завантаження та прикріплення файлів.
