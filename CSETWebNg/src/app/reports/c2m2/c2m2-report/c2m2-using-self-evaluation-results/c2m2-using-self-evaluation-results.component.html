5. Використання Результатів Самооцінки

Модель C2M2 призначена для використання організацією для систематичної оцінки її кібернетичних здатностей, для комунікації рівнів здатностей у значущих термінах та для інформування щодо пріоритетів інвестицій у сфері кібернетичної безпеки. Рисунок 4 узагальнює потенційний підхід до використання моделі. Організація проводить самооцінку за моделлю, використовує цю самооцінку для виявлення рівнів здатностей, пріоритизує ці рівні та розробляє плани для їхньої адресації, і, нарешті, впроваджує плани для усунення рівнів. При виконанні планів, бізнес-цілі змінюються, а середовище ризику розвивається, процес повторюється. Цей розділ надає короткий огляд того, як використовувати результати самооцінки за цим підходом. Для більш докладного огляду цих кроків і додаткових вказівок, див. розділ "Використання Моделі" документа моделі C2M2 V2.1, доступний за посиланням: https://energy.gov/C2M2.

Рисунок 4: Пропонований Підхід до Використання Моделі

Цей звіт узагальнює результати самооцінки організації, проведеної в рамках Кроку 1, Провести Самооцінку.

Він надає моментальне уявлення про кібернетичний стан функції, яка перебуває в області видимості. Учасники семінару самооцінки повинні розглянути цей звіт та спільно вирішити всі невідповідності або питання до наступного кроку.

На Кроці 2, Проаналізувати Виявлені Рівні, організація виявляє рівні виконання модельних практик, оцінюючи результати самооцінки на відповідність цільовому профілю - бажаному профілю, що представляє рейтинг цілі для кожного домена в моделі. Організації, які використовують модель вперше, можуть визначити цільовий профіль після проведення самооцінки, в то час як інші часто визначають цільовий профіль перед проведенням самооцінки. Для отримання додаткової інформації щодо встановлення цілей, див. Додаток D, "Встановлення Цілей" у Посібнику з Самооцінки C2M2, доступному за посиланням: https://energy.gov/C2M2.

На Кроці 3, Пріоритизувати та Розробити Плани, організація використовує аналіз рівнів для пріоритизації дій, необхідних для повного впровадження практик у цільовому профілі. Аналіз вартості та користі може допомогти у визначенні пріоритетів необхідних дій. Організація має розробити план для вирішення обраних рівнів та призначити власника плану, який має достатню повноваження для нагляду за його впровадженням.

Організаційне керівництво повинно регулярно проводити перегляди для оцінки стану, вирішення перешкод та виявлення необхідних корекцій під час впровадження.

На Кроці 4, Впровадити Плани та Періодично Переоцінювати, плани, розроблені на попередньому кроці, повинні бути впроваджені для вирішення виявлених рівнів. Послідовні самооцінки моделі особливо корисні для відстеження впровадження та повинні проводитися періодично для забезпечення досягнення бажаного прогресу. Переоцінки також слід розглядати в разі великих змін в бізнесі, технології, ринку або середовищі загроз для того, щоб забезпечити, що поточний профіль відповідає бажаному стану організації.
