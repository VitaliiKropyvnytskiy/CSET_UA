Compare

 
Once the assessment file table is complete, click the Next button to begin the Compare analysis.
 
Compare Results: The analysis for a Compare aggregation presents several graphs. This analysis is more complex than that for Trend. The Compare screen has additional tabbed headings. These are Overall Summary, Commonly Missed Questions, Individual Assessment Comparison, and Best to Worst.
 
An HTML report may be created by clicking the Generate Compare Report button. The report will include the information from the analysis screens and the Mode and SAL from the default assessment.
 
Overall Summary: This tab is opened by default and presents compliance information by standard versus component and by security category.

Commonly Missed Questions: This tab presents a table of missed questions from all the assessments analyzed listed by category and subcategory. A missed question is one either answered No. Answers of Yes, N/A, ALT, or Unanswered are not included in the table.
 
 Figure: Compare Commonly Missed Questions tab
 
Individual Assessment Comparison: This screen is similar to the Overall Summary, except that it breaks out the information by assessment. A table shows the total answers given by assessment. Next, a chart displays compliance by components and standards, by assessment. The last graph on the left provides the SAL of each assessment to aid in identifying the high-risk facilities.

Best to Worst: This screen requires the user to select a topic on the left before the graph on the right is populated. The figure below provides an example with Account Management selected.
 
Figure: Compare Best to Worst
 
The graph presents the percent of each answer option by assessment. The bars are ordered to show the most compliant assessment at the top and the least compliant at the bottom. The best is considered to have the largest number of Yes and ALT answers among the included assessments.
