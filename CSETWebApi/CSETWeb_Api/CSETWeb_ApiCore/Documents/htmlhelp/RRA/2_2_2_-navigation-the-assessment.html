Найменування практики

 

 
Ідентифікатор цілі

 

 

Кожне запитання (ціль) містить ідентифікатор, структурований наступним чином:

 

Ціль [GG]:Рівень [B,I,A].Номер запитання [Q##]

Наприклад: AM:А.Q03

•ціль: АМ - Управління активами

•рівень: високий

•запитання: №4

 
Рівень практики

 

Вказує на відповідний рівень.

 
Піктограми (іконки) CSET

 

У розділі "2.2.3. Компоненти RRA оцінювання у CSET" наведено  детальний опис кожної піктограми CSET, а також інформацію про результат активації дій по кожній з них, які здійснюються користувачем під час проведення оцінювання.

 
Варіанти відповідей на запитання

 

 

Результат оцінювання  представлений як ТАК: організація повністю виконує діяльність, зазначену в запитанні.

Для кожної практики в опитувальнику можуть включатися твердження, які вказують, що відповідь відповідає практиці, і документуються відповідні докази, які підтверджують цю відповідь.

Для результату опитування Так експерт може активувати дії по піктограмі  "Спостереження" (див. "2.2.3. Компоненти RRA оцінювання у CSET") для запису відповідних спостережень.

 

Результат оцінювання  представлений як НІ: організація не виконує діяльність у повному обсязі.

Для кожної такої практики експерт додає пояснення про те, чому практика не виконується, а також відповідне документальне підтвердження.

Для результату опитування Ні експерт може активувати дії по піктограмі  "Спостереження" (див. "2.2.3. Компоненти RRA оцінювання у CSET") для запису відповідних спостережень..

Результат опитування відмічений позначкою "Для перегляду": запитання вимагатиме подальшого перегляду та вивчення.

 
Індикатор виконання

Індикатор виконання стане синім і замкнеться в коло по мірі того, як користувач відповідатимете на запитання в кожній практиці.

Після завершення процесу оцінювання на Індикаторі виконання з'явиться прапорець (P).
